
# Blog app

A blog website where user can publish their blogs


## Features

- Add and remove blog
- profile option
- Authentication
- user management
- User profile Edit options


## Run Locally

Clone the project

```bash
  git clone https://github.com/Anoosh-MR/Blog-app.git
```

Go to the project directory

```bash
  cd Blog-app
```

Install dependencies

```bash
cd api
    npm install

cd  client
    npm install
```

Start the server
```bash

cd client && cd api
  npm  start

  ```
